A example

tr类型能增加td类型的子元素

  1. 申明一个变量；通过一个找/新建id的方法
    var userInfo = document.getElementById(“userInfo”);
    var row = document.createElement(“tr”);
  2. 给变量赋值；通过获取id/变量名的值
    td_unitprice.innerHTML= $(“productunitprice”).value;
  3. 把变量加入父级
    row.appendChild(td_id);
